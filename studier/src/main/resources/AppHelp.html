The Application Help

The bible reader page

On the first page displayed you can read the word. Here on the left there is a tree-view to navigate through the books and chapters.

The buttons <-- and --> are for paging through the chapters either forward or backward.

There is a control there the visited chapters are in a history. Another control is to selectt dictionaries which are if selected displayed in another window.

On the left there is a list-box containing the bible translations you loaded you are able to switch here between the translations. The righzt mouse - click 

pops up a context menu for calling the bible information dialog. 

The area and the functionality

In the area you can select verses by click and then with the right click you get a context menu there you can select to store the verse as note or as highlight. As well the verse can be copied for usage then editing a devotional. Like e.g.

[1. Mose 1,17]

[1. Mose 1,17]: 17. And God set them in the firmament of the heaven to give light upon the earth,

If there is a dictionary (accordance) open you can select text in the usual way and then start a search by right mouse-click and click the right menu item.

In the listbox on bottom there are study notes. If this notes contain 'links' to verses you can select them by selecting the listbox-item. They are then displayed in a dialog there you can click on the verse as link and open the full chapter.
